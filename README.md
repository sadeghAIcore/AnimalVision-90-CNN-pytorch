# AnimalVision-90-CNN-pytorch

AnimalVision-90-CNN-pytorch is a deep learning image classification project built with **PyTorch** that can recognize **90 different animal classes** from input images.

This project uses a convolutional / transfer learning-based approach for animal image classification and is designed to be simple, practical, and easy to extend.

---

## Features

- Classifies **90 animal categories**
- Built with **PyTorch**
- Supports image inference on custom inputs
- Trained for multi-class animal recognition
- Can be deployed or shared through **Hugging Face**
- Easy to modify and improve for new datasets

---

## Project Overview

The goal of this project is to identify animals from images using a deep learning model trained on 90 classes.

The model takes an input image, preprocesses it, runs inference, and returns prediction probabilities for the animal classes.  
This makes it useful for:

- learning PyTorch image classification workflows
- experimenting with transfer learning
- building animal recognition demos
- deploying a trained classifier online

---

## Tech Stack

- **Python**
- **PyTorch**
- **Torchvision**

## Model Capability

This model is able to classify images into **90 different animal categories**.

Depending on the training setup and dataset quality, performance may vary across classes, but the system is designed to provide fast and practical animal recognition using deep learning.
